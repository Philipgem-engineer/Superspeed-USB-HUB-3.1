# SUPERSPEED USB 3.1 HUB (5 Gbps)
This reposistory containts a schematic and PCB design of a superspeed USB 3.1 Hub (5 Gbps)

##  FEATURES
- 5 Gbps Speed
- Can charge devices up to 7.5 watts
- Type C upstream port
- 4 USB-A downstream port
